# Contributing to Project Genesis

Thanks for your interest in contributing! We welcome contributions from the community.

How to contribute

1. Fork the repository.
2. Create a branch: git checkout -b feature/your-feature
3. Make your changes and add tests where appropriate.
4. Run linters and tests.
5. Create a pull request describing your changes.

Guidelines

- Keep changes focused and small.
- Write clear commit messages.
- Add/update tests for new behavior.
- Follow the code style (ESLint, Prettier).

Code of Conduct

By contributing you agree to follow the project's Code of Conduct.

Security

If you discover a security vulnerability, please report it privately as described in SECURITY.md.
