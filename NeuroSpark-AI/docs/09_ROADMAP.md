🚀 Additions considered core to the project

These are not random ideas, but features that could add substantial product value if implemented well:

1. Potential Timeline

Shows how a user's abilities changed over time instead of static scores.

2. Skill Tree

An RPG-like skill tree where users see developed skills and what is needed to unlock new levels.

3. Mission Generator

Automatically generates new challenges tailored to the user's level and interests.

4. Career Simulator

Presents a virtual day in a specific career, asks decisions, and uses them in assessment.

5. Evidence Center

For every recommendation, users can view the evidence behind it (e.g., challenge results), increasing transparency.

6. Ethics & Transparency

A page explaining that results are estimates based on data and not absolute judgments.
