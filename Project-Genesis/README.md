# Project Genesis

> Discover. Develop. Become.

Project Genesis is an AI-powered platform designed to help people discover their strengths through practical challenges, cognitive assessments, and continuous progress tracking.

Unlike traditional personality tests, Project Genesis builds an evidence-based profile over time using user performance rather than one-time answers.

---

## Vision

Help every person understand what they are naturally good at and provide a clear roadmap for continuous improvement.

## Confidence Score (New)

Reports include both a score and a "confidence level" that describes how much data supports the assessment (e.g., Low, Medium, High). As users complete more challenges and assessments, confidence levels rise, making recommendations more reliable.
